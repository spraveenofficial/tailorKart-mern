## Features

- Landing Home Page
- Category Wise Products
- Filter Products
- Each Product Page
- Login / Signup Page
- Cart Management
- Wishlist Management
- Review Order
- Add Address
- JWT based Authentication

---

## Built with

- React JS (v17.0.2)
- Componently Component Library
- React Router v6
- Backend made Using ExpressJS & MongoDB (NodeJS)
